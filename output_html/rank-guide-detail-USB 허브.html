USB 허브 구매 가이드

1. 다양한 포트

  • usb 메모리, 무선 키보드, 마우스, 프린터 등 usb 포트 위주로 사용할 목적이라면 usb 포트 수가 많은 제품이 유용합니다
  • 하지만 SD카드나 micro SD 카드를 지원한다면 외부에 있을 때 고프로나 블랙박스 등을 확인할 수 있어 활용성이 좋습니다

2. 케이블 길이

  • 반면 데스톱에 연결해 책상 위에 두고 사용하는 경우라면 여유 있게 사용할 수 있는 1m 이상의 넉넉한 케이블 길이를 가진 usb허브 제품을 추천합니다
  • 노트북에 사용할 경우에는 케이블 길이가 길면 거추장스러울 수 있기 때문에 케이블 길이가 짧은 제품이 편리합니다

3. 부가기능

  • 개별 on/off 버튼이 있는 경우 전력을 아끼고 안전하게 사용할 수 있습니다
  • LED 표시등은 작동 여부를 알 수 있어 편리합니다
  • 스마트폰에서 보이는 화면을 TV나 모니터 등에 복제하여 출력하는 미러링 기능이 있다면 외부 영상 출력이 가능합니다

1. 벨킨 USB C타입 노트북 멀티미디어 허브

💡 애플 유저에게 추천

최저가 보기

  • 크기가 작아 휴대성이 좋지만 usb3.0 연결 시 너무 빡빡하다는 후기를 볼 수 있습니다
  • 최대 5Gbps 전송 속도와 최대 60W 전력 전달과 다양한 포트 확장을 동시에 지원합니다
  • SD 카드 이동 속도가 느려 SD카드 슬롯이 필요해서 사용하실 분께는 추천하지 않는다는 리뷰가 있으니 구매에 참고해 주세요
  • 10만 원대로 가격이 다소 높지만 벨킨 공식 수입사 인증 정품으로 믿고 구매할 수 있는 usb 허브 추천 제품입니다
  • 애플 기기를 사용한다면 이 제품을 추천한다는 리뷰가 있으며 맥북 유저가 빔프를 사용하고 싶다면 좋은 선택이 될 수 있다는 추천글이 있습니다
  • 또한 삼성 덱스 모드 지원으로 삼성 스마트폰을 pc처럼 활용할 수 있으며 2년 무상 품질 보증을 제공하고 있습니다
  • 아이패드 프로 3, 4세대, 맥북, USB-C PD 호환 노트북에 사용 가능하며 아이패드 프로 3를 충전하면서 동시에 데이터 고속 전송, 모니터와 유선 인터넷 연결이 가능합니다
  • SD카드 전송 속도가 느리긴 하지만 안정적으로 사용할 수 있어 만족스럽다는 후기도 있습니다
  • 맥북 또는 애플 기기를 사용하는 유저라면 벨킨 멀티미디어 허브 F4U092btSGY를 주목해 주세요

  • 애플 유저에게 추천

최저가 보기

  • 맥북에 연결하는 c타입 단자가 너무 뻑뻑해서 탈착 시 힘을 강하게 줘야 하는 점을 단점으로 꼽은 리뷰가 있으니 구매에 참고해 주세요
  • SD카드/ C 타입/ HDMI/ USB 인식 모두 잘 되며 가격대도 저렴한 거 같아 가성비가 좋다는 의견을 남겨주셨습니다
  • 다양한 기기와 연결해 사용할 수 있는 usb 허브를 찾고 계신 분께 좋은 선택지가 될 수 있는 제품입니다
  • HDMI 화질도 괜찮고 발열도 없어 만족스럽다는 후기가 있는 반면 발열이 다소 있다는 의견도 있어 발열에 있어서는 호불호가 있는 편입니다
  • 가격 대비 마감이 준수하고 PD 전력이 높으며 비싼 도킹 제품 구매할 거 아니면 이 모델을 추천한다는 후기가 있습니다
  • 삼성 덱스 지원/ 닌텐도 스위치 호환/ 스마트폰, 노트북 화면을 고화질 대형 TV 화면으로 출력해 사용하고 싶으신 분들께 추천하는 usb 허브입니다
  • 7 in 1 무전원 타입 C 멀티포트로 usb 이외에도 다양한 형태의 멀티 단자를 사용할 수 있으며 c type PD 고속 충전이 가능합니다

  • 각 포트별 전원 온/오프가 가능하여 전력 소비량이 많은 제품도 사용하기 좋으며 바닥에 미끄럼 방지 처리가 되어 있는 점도 마음에 든다는 상품평이 있습니다

  • 첫 번째로 소개해드리는 usb 허브 추천 제품은 공유기 이름으로도 잘 알려져 있는 ip TIME에서 출시된 ipTIME 5포트 유전원 USB3.0 허브입니다
  • 본체랑 주변기기가 멀어서 usb 허브를 찾다가 구매했으며 3.0까지 사용 가능한 usb 포트 4구와 충전 전용 포트 1구가 있으며 유전원 제품이지만 전원 연결은 선택이라는 리뷰를 남겨주셨습니다

4. 엘레컴 기특한 USB허브 U2H-SN4BF-G

💡 높은 가성비

최저가 보기

  • 약 7cm의 케이블 길이로 데스크톱 앞면 또는 노트북 등에 사용하기 적당한 usb 허브입니다
  • 디자인과 실용성에 있어서 만족스러운 제품으로 노트북이나 본체에 바로 꽂아 사용할 수 있어 편리하다는 상품평을 볼 수 있습니다
  • 엘레컴 U2H-SN4BF-G 제품은 8천 원대의 저렴한 가격대와 귀여운 이모티콘으로 포인트를 준 usb 허브 추천 제품입니다
  • 차량에서 사용하기 위해 구매했으며 작동 잘 되며 얼굴 표정이 그려져 있어서 귀엽다는 리뷰가 있습니다
  • 가격대도 저렴하면서 노트북에 사용 시 걸리적거리지 않아서 편의성에 있어서 대만족이라는 후기를 남겨주셨습니다
  • 콤팩트한 스틱 타입으로 휴대 및 사용이 편리하며 허브 하나로 4개의 포트를 사용할 수 있어 유용합니다
  • 드라이버 설치 없이 연결해 자동으로 연결 가능하며 USB 2.0 하이 스피드 모드를 지원해 빠른 속도를 제공합니다

장점

  • 높은 가성비
  • 뛰어난 실용성
  • 높은 상품평

단점

  • 케이블 길이가 다소 짧다는 후기
